# META PROMPT GENERATOR

Изучи исходный пользовательский запрос:

```
$ARGUMENTS
```

Выполни следующие задачи строго по шагам:

1. **Переформулировка запроса.**  
   Переформулируй этот запрос так, чтобы он соответствовал лучшим практикам prompt-инжиниринга:
   - максимальная ясность и конкретность,
   - наличие цели, контекста и ограничений,
   - чёткие критерии качества результата,
   - явный формат вывода,
   - пошаговая структура рассуждений.
   
2. **Уточнение деталей**
   - Выдели и уточни у пользователя любые неясные или неоднозначные моменты в исходном запросе.
   - Ставь все что говорит пользователь под сомнение, критически и непредвзято оценивай утверждения.
   - Рассмотри задачу с разных точек зрения, задавая наводящие вопросы которые могут помочь сформировать контекст задачи лучше.

3. **Создание плана решения.**  
   На основе улучшенного запроса сформируй подробный план действий:
   - что нужно проанализировать,
   - какие шаги предпринять,
   - какие параметры, условия, ограничения и критерии должны учитываться.

4. **Генерация нескольких равнозначных вариантов решений.**  
   Создай минимум три независимых варианта решения задачи на основе созданного плана.  
   Каждый вариант должен быть полностью самодостаточным и описывать свой уникальный подход.  
   Не указывай, какой из вариантов лучше — все они равнозначны.

5. **Контекст и цель.**  
   Добавь развернутый контекст задачи, включая:
   - описание проблемы,
   - приоритеты,
   - общие принципы улучшения исходного запроса,
   - объяснение, почему предложенные пути решения покрывают проблему.

6. **Делегирование анализа в codex 5.1.**  
   Передай всю итоговую структуру (улучшенный запрос, план, варианты решений и контекст) как единый пакет информации в модуль анализа **codex 5.1** и потребуй, чтобы он:
   - применил критическое мышление и поставил под сомнение каждое из предложенных решений,
   - непредвзято проверил логичность, полноту и состоятельность всех вариантов,
   - выявил возможные логические ошибки, скрытые допущения и конфликтующие выводы,
   - применил этическое мышление (влияние на людей, риски, потенциальные негативные последствия),
   - опирался на лучшие практики анализа, верификации и ревью решений,
   - структурировал результат анализа в виде:
     - замечаний по каждому варианту,
     - общих рекомендаций по укреплению решений,
     - предложений по комбинированию или улучшению подходов,
     - **рейтинга каждого решения по шкале от 1 до 10**,
     - **определения, какой вариант является предпочтительным с точки зрения анализа codex 5.1** и почему.

7. **Финальный результат.**  
   На основе ответа от **codex 5.1** предоставь итоговую сводку:
   - оценку качества всех решений,
   - сильные стороны каждого подхода,
   - возможные улучшения,
   - сравнительный вывод,
   - **итоговый выбор предпочтительного варианта**,
   - **сводный рейтинг по каждому из вариантов**,  
   - обоснование, насколько хорошо весь набор решений покрывает исходную проблему.

Выводи результат строго в структурированном многоуровневом формате.
