* üöÄ Quick Start

** Prerequisites

macOS (Apple Silicon or Intel)

** Bootstrap Installation

The system uses *Nix* as the foundation for reproducible package management. Install it first:

#+NAME: install-nix
#+BEGIN_SRC bash :tangle bootstrap.sh
# Install Nix package manager (multi-user installation)
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | \
  sh -s -- install

# Source nix
. /nix/var/nix/profiles/default/etc/profile.d/nix-daemon.sh
#+END_SRC

** Clone Configuration

#+NAME: clone-config
#+BEGIN_SRC bash :tangle bootstrap.sh
# Backup existing config
[ -d ~/.config ] && mv ~/.config ~/.config.backup.$(date +%Y%m%d_%H%M%S)

# Clone dotfiles
git clone https://github.com/Artawower/.config.git ~/.config
cd ~/.config
#+END_SRC

** Initialize with Nix

#+NAME: nix-init
#+BEGIN_SRC bash :tangle bootstrap.sh
# Build and activate the configuration
cd ~/.config/nix
nix flake update  # Update dependencies

# For Darwin (macOS) system configuration
nix build .#darwinConfigurations.$(hostname -s).system
./result/sw/bin/darwin-rebuild switch --flake .

# For Home Manager
nix run home-manager/master -- switch --flake ~/.config/nix
#+END_SRC

** Post-Installation

#+NAME: post-install
#+BEGIN_SRC bash
# Install launch agents for system theme watching
cd ~/.config/launchagents
./install.sh

# Configure Xonsh as default shell
echo "$(which xonsh)" | sudo tee -a /etc/shells
chsh -s $(which xonsh)

# Restart terminal
#+END_SRC
